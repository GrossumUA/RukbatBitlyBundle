RukbatBitlyBundle
=================

Symfony2 bundle for the `bitly <http://dev.bitly.com/api.html>`_ API.
This bundle wraps the `Falicon/BitlyPHP <https://github.com/Falicon/BitlyPHP>`_ library in a Symfony2 bundle.
